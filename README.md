# 🌿 Remèd Lakay

**Remèd Lakay** se yon asistan AI pwofesyonèl pou medsin tradisyonèl ayisyen. Li fèt pou ede moun jwenn enfòmasyon sou plant medsin yo nan fason ki senp epi efikas.

## ✨ Fonksyonalite
- **Asistan AI (Konpè Fèy)**: Yon ekspè nan medsin fèy ki reponn keksyon w yo an kreyòl.
- **Enstalasyon PWA**: Ou ka enstale aplikasyon an sou telefòn ou tankou yon vrè app.
- **Bwat Sekirite**: Sistèm aktivasyon ak kòd pou pwoteje kontni an.
- **Baz Done Foto**: Deteksyon otomatik ak afichaj foto plant medsin yo.
- **Konvèsasyon Pèsistan**: Mesaj ou yo toujou rete sove sou aparèy ou.

## 🚀 Kouman pou w kòmanse
1. **Enstale depandans yo**:
   ```bash
   npm install
   ```
2. **Konfigirasyon**:
   Kreye yon fichye `.env` epi mete kle sa yo:
   ```env
   CHATPDF_API_KEY=kle_api_ou
   CHATPDF_SOURCE_ID=cha_xxxxxxxxxx
   ```
3. **Lanse aplikasyon an**:
   ```bash
   npm run dev
   ```

## 🛠️ Teknoloji ki itilize
- **Frontend**: React + Vite
- **Styling**: Tailwind CSS / Vanilla CSS
- **AI**: ChatPDF API
- **PWA**: vite-plugin-pwa
